# Chapter 1: Tải R xuống và cài đặt vào máy tính

## Core Idea
Hướng dẫn cách tải phần mềm R từ mạng CRAN và cài đặt lên máy tính cá nhân (Windows).

## Frameworks Introduced
- **CRAN (Comprehensive R Archive Network)**: Mạng lưới máy chủ toàn cầu lưu trữ mã nguồn và các bản cài đặt của R.

## Key Concepts
- **CRAN**: Website chính thức để tải R (http://cran.R-project.org).
- **R version**: Phiên bản của phần mềm R (ví dụ: R-2.2.1-win32.exe).
- **Setup**: Quá trình cài đặt R chỉ mất khoảng 1 phút.
- **R icon**: Biểu tượng R trên desktop để khởi động chương trình.

## Mental Models
- **R as a Workbench**: Cài đặt R là bước đầu tiên để xây dựng môi trường phân tích dữ liệu chuyên nghiệp.

## Anti-patterns
- **Tải từ nguồn không chính thức**: Luôn sử dụng CRAN để đảm bảo an toàn và cập nhật.

## Code Examples
```bash
# Không có code R trong chương này, chủ yếu là thao tác trên web và Windows.
# Link tải ví dụ:
http://cran.r-project.org/bin/windows/base/R-2.2.1-win32.exe
```

## Worked Example
1. Truy cập `http://cran.R-project.org`.
2. Chọn "Download R for Windows" -> "base".
3. Tải file `.exe` (ví dụ: `R-2.2.1-win32.exe`).
4. Chạy file đã tải, nhấn Next cho đến khi hoàn tất.
5. Click vào icon R trên Desktop để bắt đầu.

## Key Takeaways
1. R là phần mềm hoàn toàn miễn phí và mã nguồn mở.
2. Việc cài đặt cực kỳ đơn giản và nhanh chóng.
3. Luôn kiểm tra phiên bản mới nhất trên CRAN.
