# Tidyverse & R4DS Glossary

- **Aesthetic Mapping (aes)**: Defines how data variables map to visual properties (color, shape, x/y coords) in a ggplot2 layer.
- **dplyr**: Tidyverse package for data manipulation built around six key verbs (`filter`, `arrange`, `distinct`, `mutate`, `select`, `summarize`).
- **Faceting**: Plot splitting method that creates multiple panels (small multiples) based on levels of a categorical variable.
- **forcats**: Tidyverse package designed specifically for manipulating categorical factors.
- **ggplot2**: Tidyverse data visualization package based on the Grammar of Graphics.
- **lubridate**: Tidyverse package simplifying date-time parsing and arithmetic.
- **Pipe (|>)**: The native R operator that chains functions by passing the left-hand result as the first argument of the right-hand function.
- **readr**: Tidyverse package for fast and user-friendly reading of rectangular flat files (CSV, TSV).
- **Tibble**: A modern, lazy re-imagining of R's standard `data.frame` that prints cleanly and does not implicitly rename columns.
- **Tidy Data**: A consistent data layout where each variable is a column, each observation is a row, and each cell contains a single value.
- **tidyr**: Tidyverse package dedicated to reshapping and cleaning data structures (pivoting).
- **YAML**: "YAML Ain't Markup Language" header format used to define metadata and output parameters in Quarto reports.
